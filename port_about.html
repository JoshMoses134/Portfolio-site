This is a WordPress quiz plugin with the conditional logic feature where the next question in the quiz depends on user's answer to the previous question. This allows you to create highly interactive personality quizzes that change dynamically. It supports multiple choice, single choice, and essay answers. You can assign scoring to each correct answer and decide what to do when the user selects an answer.
